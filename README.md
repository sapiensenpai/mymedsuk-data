# MyMeds UK Data Repository

This repository contains the MHRA medication database for the MyMeds UK application.

## Files

- **mhra_ultra_3.0.json** (53MB) - Main hierarchical database with 3,716 substances and 27,848 products
- **all_pdf_links.json** (46MB) - Flat list of all 66,732 PDF documents
- **mhra_structure_mapping.json** (5.3MB) - Folder structure mapping
- **update_certificate.json** - Database version and statistics

## Usage

These files are served via GitHub Pages for the MyMeds UK Framer application.

### Endpoints

Once GitHub Pages is enabled, the files will be available at:

- https://sapiensenpai.github.io/mymedsuk-data/mhra_ultra_3.0.json
- https://sapiensenpai.github.io/mymedsuk-data/all_pdf_links.json
- https://sapiensenpai.github.io/mymedsuk-data/mhra_structure_mapping.json
- https://sapiensenpai.github.io/mymedsuk-data/update_certificate.json
